resize()method
public static function resize(bmpd:BitmapData, width:Number, height:Number, keepAspectRatio:Boolean = true, pixel:Boolean = false, transparent:Boolean = true):BitmapData

Resize l'image en conservant, ou non, l'aspect initial.

Parameters
bmpd:BitmapData — BitmapData L'image à resizer.
 
width:Number — Number La largeur après resize.
 
height:Number — Number La hauteur après resize.
 
keepAspectRatio:Boolean (default = true) — Boolean Si oui ou non l'aspect de l'image est conservé.
 
pixel:Boolean (default = false) — Boolean Si oui ou non les pixels grossissent, et donc que l'image est déformée lorsque la largeur et la hauteur voulue sont plus grandes que la taille de l'image.
 
transparent:Boolean (default = true) — Boolean Si oui, le BitmapData renvoyé sera transparent.

Returns
BitmapData
