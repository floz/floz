Constructor detail
MassLoader()constructor
public function MassLoader(isPersistent:Boolean = false)

Creates a loader that can load a list of files. To load an unique file, this loader is not the best way, you should refer to ImageLoader or TextLoader class. Call the addItem method or addItems method to add url to load. Then, call the loadNext method to load the files by order.

Parameters
isPersistent:Boolean (default = false) — Boolean If false, all the data loaded will expire after the call of the destroy method.

Destroy the loader to save memory. If the isPersistent variable is false, no more data will be linked to the loader. After the call of this method, the loader will be unable to use.
