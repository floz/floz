public function MovieLoader(isPersistent:Boolean = false)

Creates a loader that can load movie files such as SWF, PNG, JPG... To load text file, you must use the TextLoader object. Call the load() method and pass an url as parameter to launch the loader.

Parameters
isPersistent:Boolean (default = false) — Boolean If false, all the data loaded will expire after the call of the destroy method.

public override function destroy():void

Destroy the loader to save memory. If the isPersistent variable is false, no more data will be linked to the loader. After the call of this method, the loader will be unable to use.
